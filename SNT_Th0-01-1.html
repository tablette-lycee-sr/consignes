ÉTAPES


01-1. Les instructions

Leçon
- Un programme est composé d'une suite d'instructions. Chaque instruction est une action que le programme doit exécuter.
- On rédige généralement une seule instruction par ligne de code. Celles-ci sont lues l'une après l'autre, de haut en bas, de la première à la dernière ligne. On peut saisir une à une des instructions dans la fenêtre Console, mais si l'on veut enregistrer toutes les actions dans un programme, on les rédige dans un script que l'on sauvegarde.

- Ensuite ce script doit être converti en langage machine pour pouvoir être exécuté par l'ordinateur. En python, le code source du script est traduit par un interpréteur qui produit le code binaire que seul l'ordinateur sait lire.
